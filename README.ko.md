# RE Tension Game Making

*English: [README.md](README.md)*

**2분 안에 최대 점수를 노리는 웹 1인칭 슈터(2.5D 레이캐스터)** 게임입니다.  
적을 빠르게 처리하고, 끝난 뒤 닉네임으로 점수를 등록해 랭킹에 도전하세요.

## 게임 개요

- 장르: 하이퍼 캐주얼 FPS (2.5D 레이캐스터)
- 플레이 타임: 1판 약 2분
- 목표: 제한 시간 내 처치 점수 최대화
- 점수 규칙: 적 처치 시 점수 획득
- 종료 조건: 시간 종료 또는 HP 0

## 조작법

### 데스크톱

- 이동: `WASD` 또는 방향키 (`IJKL`도 지원)
- 시점 회전: 마우스 이동 (포인터 락)
- 발사: 마우스 클릭 또는 `Space`

### 모바일

- 터치 상단: 전진
- 터치 하단 왼쪽/오른쪽: 좌/우 회전
- 두 손가락 터치: 발사

## 게임 흐름

1. 타이틀 화면에서 시작
2. 2분 동안 전투 진행 (HUD: 점수, HP, 타이머)
3. 게임 종료 후 닉네임 입력
4. 리더보드 점수 등록

## 빠른 실행

```bash
npm start
# 또는 npm run dev
```

- 기본 주소: `http://localhost:3000`
- 스크립트는 `web/` 디렉터리를 `serve`로 실행합니다.

## 프로젝트 구조

- `web/`: 실제 게임 코드 (`index.html`, `src/main.js` 등)
- `docs/`: 실행/공유 관련 문서
- `Temp/`: 참고 자료 보관

## 같은 네트워크에서 접속

같은 Wi-Fi의 다른 기기에서 접속하려면 `http://<내-로컬-IP>:3000` 형태로 접속하세요.  
방화벽, 네트워크 바인딩, 트랙패드 이슈는 `docs/local-lan-sharing.md`를 참고하세요.

## 개발 정보

- 환경 변수 및 API 키(`.env` 등)는 커밋하지 않습니다.
- 에이전트 작업 규칙은 `CLAUDE.md`, `AGENTS.md`를 참고하세요.
